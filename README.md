# 🎓 Student Management System

A simple and efficient **Student Management System** developed in **Python**. This is a menu-driven console application that helps manage student records, including adding, updating, searching, viewing, and deleting student information. The project stores data using a JSON file, making it lightweight and easy to use.

## 📌 Features

- Add new student records
- View all students
- Search students by Roll Number
- Update student details
- Delete student records
- Automatic grade calculation based on marks
- Store student data using JSON
- User-friendly menu-driven interface

## 🛠️ Technologies Used

- Python 3
- JSON
- File Handling
- Functions
- Loops
- Conditional Statements
- Exception Handling

## ▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

```bash
python student_management_system.py
```

5. Follow the on-screen menu to perform different operations.
